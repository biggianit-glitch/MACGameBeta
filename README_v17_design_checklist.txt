MACTheGame v17 Design Coverage Checklist

Covered from original discussion:
- Browser-based HTML/JavaScript game runnable through VS Code / Live Server.
- Retro platformer format inspired by classic 8-bit/16-bit structure.
- Character/profile selection using MAC personas.
- Starting skill selection tied to “Excel at Your Role.”
- Career Journey Map with level nodes, completion checkmarks, and manual level selection.
- 10 themed Amneal/MAC career levels.
- Four major skill collectibles per level.
- Coin-style frequent rewards.
- Timer, tries, score, skill counter, and level/world display.
- Restart current level on failure, no mid-level checkpoints.
- Elevator/end-of-level interaction using Up.
- Jump separated to Spacebar.
- Gig / Mentor bonus doors in different positions by level.
- Bad guys with multiple variations, patrol movement, falling into pits, and respawn.
- Career Boost Star power-up with color flicker/invincibility behavior.
- Pause screen with Continue and Main Menu.
- On-screen 3-initial leaderboard entry, no browser prompt.
- Music by screen type and level; sound effects for key actions.

v17 specific:
- Added vertical reachability helpers around high platforms.
- Added platform reachability audit.
- Reduced world info duplication in HUD.
- Added stronger 16-bit style tile shading and foreground polish.
- Updated music engine with melody, harmony, and bass pulse.
